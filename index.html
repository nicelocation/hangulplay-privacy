1. 개인정보 처리방침의 목적

본 개인정보처리방침은 한글놀이 자모쓰기(이하 “앱”)가 사용자의 정보를 어떤 방식으로 처리하는지, 사용자가 등록한 단어와 사진이 어떻게 저장되는지, 그리고 앱에서 사용하는 외부 SDK가 어떤 기술 정보를 처리할 수 있는지 설명하기 위해 작성되었습니다.

2. 앱 제공자가 직접 수집하는 개인정보

앱 제공자는 현재 사용자를 식별하기 위한 이름, 이메일 주소, 전화번호, 계정 정보, 위치정보를 직접 수집하지 않습니다. 앱은 회원가입, 로그인, 광고 표시, 위치 기반 기능, 개발자 서버 백업 기능을 제공하지 않습니다.

3. Google ML Kit 사용 및 기술 정보 처리

앱은 손글씨 인식 기능을 제공하기 위해 Google ML Kit의 기기 내 인식 기능을 사용할 수 있습니다. 손글씨 입력 내용과 인식 결과는 원칙적으로 기기 내에서 처리되며, 앱 제공자의 서버로 전송되지 않습니다.

다만 Google ML Kit SDK는 모델 다운로드, 오류 수정, 하드웨어 가속기 호환성 확인, 성능 개선, 오남용 탐지, 진단 및 사용 분석을 위해 Google 서버와 통신할 수 있습니다. 이 과정에서 기기 정보, 앱 정보, 기기 또는 기타 식별자, 성능 정보, 이벤트 유형, 오류 코드, API 구성, 기능 버전, 입력·출력 크기, 설정 언어 등 일부 기술 정보가 처리될 수 있습니다.

Google ML Kit의 데이터 처리에 관한 자세한 내용은 아래 문서를 참고할 수 있습니다.

  • Google ML Kit Terms & Privacy
  • Google ML Kit Android Data Disclosure
  • Google 개인정보처리방침

4. 개인정보의 이용 목적

앱은 기기 내부에 저장되는 정보를 다음 목적을 위해 사용합니다.

  • 유아 한글 학습용 단어 목록 표시
  • 등록한 단어와 연결된 사진 표시
  • 학습 단계, 별 보상, 보호자 설정 등 앱 사용 상태 저장
  • 사용자가 직접 만든 학습 자료를 기기 안에서 다시 불러오기
  • 손글씨 인식 기능 제공

5. 개인정보의 보관 및 파기

사용자가 앱 안에서 등록한 단어, 사진, 학습 설정 및 학습 진행 정보는 기기 내부 저장소에 보관됩니다. 사용자가 앱 안에서 해당 데이터를 삭제하거나 앱을 삭제하면 기기에서 함께 삭제될 수 있습니다.

앱 제공자는 해당 데이터를 별도의 서버에 저장하지 않으므로, 서버에 보관된 개인정보를 별도로 파기하는 절차는 발생하지 않습니다. 사용자가 직접 내보낸 백업 파일은 사용자가 저장한 위치에서 직접 삭제해야 합니다.

6. 개인정보의 제3자 제공 및 처리 위탁

앱 제공자는 사용자가 등록한 단어, 사진, 학습 기록 등 앱 안의 학습 자료를 제3자에게 판매하거나 제공하지 않습니다. 또한 해당 학습 자료를 처리하기 위해 별도의 개발자 서버나 외부 업체에 위탁하지 않습니다.

다만 제3조에서 설명한 바와 같이, Google ML Kit SDK가 기능 제공과 품질 개선을 위해 일부 기술 정보를 Google 서버와 통신할 수 있습니다. 이 기술 정보의 처리는 Google의 약관 및 개인정보처리방침에 따릅니다.

7. 앱 권한 사용 안내

앱은 사용자가 직접 사진을 등록하거나 불러오기 위해 기기의 사진 접근 권한을 요청할 수 있습니다. 사진 접근 권한은 사용자가 선택한 사진을 앱 안에서 학습 자료로 표시하기 위한 목적으로만 사용됩니다.

사진 접근 권한을 허용하지 않아도 기본 학습 기능은 사용할 수 있으나, 사용자가 직접 사진을 등록하는 일부 기능은 제한될 수 있습니다. 사용자는 기기의 운영체제 설정에서 앱의 사진 접근 권한을 언제든지 변경하거나 철회할 수 있습니다.

8. 광고, 위치정보, 계정 및 외부 분석 도구

앱은 현재 광고를 표시하지 않으며, 광고 식별자를 사용하지 않습니다. 또한 위치정보를 수집하지 않고, 회원가입 또는 로그인 기능을 제공하지 않습니다.

앱 제공자가 별도로 운영하는 외부 분석 도구나 자체 분석 서버는 사용하지 않습니다. 다만 Google ML Kit SDK의 기술 정보 처리는 제3조의 설명에 따릅니다.

9. 생성형 AI 이미지 사용 안내

앱에 포함된 일부 기본 학습용 이미지는 생성형 AI 도구를 활용하여 제작·편집되었을 수 있습니다. 이러한 이미지는 앱 제공자가 아동 학습에 적합하도록 검수한 뒤 사용합니다.

앱은 현재 사용자가 입력한 문장이나 사진을 이용해 앱 안에서 생성형 AI 이미지를 새로 만드는 기능을 제공하지 않습니다. 사용자가 직접 등록한 사진은 학습 자료 표시를 위해 기기 내부에서 사용되며, 앱 제공자의 서버로 전송되지 않습니다.

10. 아동의 개인정보 보호

앱은 유아의 한글 학습을 돕기 위한 앱입니다. 보호자는 앱 사용 중 아동이 직접 입력하거나 선택하는 단어와 사진, 학습 설정 및 학습 진행 정보가 기기 내부에 저장될 수 있음을 확인하고 관리할 수 있습니다.

앱은 현재 아동의 개인정보를 개발자 서버로 전송하거나 온라인 계정과 연결하지 않습니다. 보호자는 앱 안의 삭제 기능, 기기 설정의 권한 관리, 앱 삭제를 통해 기기에 저장된 정보를 관리할 수 있습니다.

11. 이용자 및 보호자의 권리

이용자 및 보호자는 다음과 같은 방법으로 앱 관련 데이터를 관리할 수 있습니다.

  • 앱 안에서 직접 등록한 단어와 사진 삭제
  • 기기 설정에서 사진 접근 권한 변경 또는 철회
  • 앱 삭제를 통한 기기 내부 저장 데이터 삭제
  • 사용자가 직접 내보낸 백업 파일 삭제

앱 제공자는 사용자의 단어, 사진, 학습 기록을 서버에 보관하지 않으므로, 서버에 저장된 해당 데이터를 조회하거나 삭제하는 절차는 제공하지 않습니다. 개인정보 처리와 관련한 문의가 필요한 경우 제13조의 문의처로 연락할 수 있습니다.
